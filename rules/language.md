# 言語ルール

## 基本方針

**すべてのコミュニケーションは日本語で行うこと。**

これは Claude (AI) がこのプロジェクトで回答・記述する際のすべての場面に適用される。

## 適用範囲

- **PR のタイトル・本文** — 必ず日本語で記述する
- **コードレビューのコメント** — 日本語で記述する
- **Issue のコメント・説明** — 日本語で記述する
- **Claude への質問・回答** — 日本語で回答する
- **エラーメッセージの説明** — 日本語で説明する

## 例外

- コードそのもの（変数名・関数名・コメント）は `rules/coding-style.md` に従う
- コミットメッセージは `rules/git.md` に従う（英語・動詞始まり）
- 外部ドキュメントや仕様書の引用部分はそのままでよい

## PR 記述例

```
タイトル: アレルゲン複数指定に対応する

## 概要
アレルゲンを複数同時に指定できるよう機能を拡張した。

## 変更内容
- `target` フィールドをカンマ区切りリストとして解析
- 各アレルゲンに異なる色のハイライトを適用

## テスト
- `go test ./...` がすべて通ることを確認
```
