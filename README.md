# aller-navi

## 開発環境構築
### Tkinterのインストール
```bash
brew install python-tk@3.12
```

### pipenvのインストール
```bash
# venvのインストール
python -m venv venv

# 仮想環境のアクティベート(Mac)
source venv/bin/activate

# 仮想環境のアクティベート(Windows)
myenv\Scripts\activate

# pipをアップグレード
python -m pip install --upgrade pip

# pipenvのインストール
pip install pipenv

# 仮想環境のアクティベート
pipenv shell
```
